About me

I am a second-year graduate student in the School of Interactive Computing at Georgia Tech, pursuing MS in Robotics. I am working on learning trait preferences from expert demonstrations for multi-agent team formation. I am also working on creating Starcraft expert demonstrations for a project which learns Gaussian processes from sub-optimal demonstrations and tries to find the best possible coalition with given set of robots. My current research also includes a project on understanding the concept of trainability of agents in a decentralized market based coalition formation framework.

I graduated from the University of Illinois Urbana-Champaign with a Bachelor’s in Mathematics and Computer Science where I developed robotic systems for a teleoperated nursing robot. I am interested in working on learning algorithms and developing middleware for autonomous robotic systems.

What i'm doing

  • Learning Trait Preferences for Task allocation

    Develop algorithms to extract the underlying preferences from expert demonstrations. Using the inferred preferences to make the multi-robot task allocation more effective.

  • Decentralized MRTA accounting for Trainability of agents

    Improving the existing decentralized MRTA method to incorporate feedback of trainability of an agent from the coalition level

  • User Study setup

    Helping setup a Starcraft user study to collect demonstrations which will be used in learning a gaussian process to improve the performance of the task allocation.

  • Personalized pose detection

    Extend the existing algorithms to learn a personalized embedding to distinguish heterogeneity from person-to-person differences for pose prediction.
